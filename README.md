# My Static Site Generator

This is a guided project from [boot.dev](https://boot.dev)

The project generates an HTML page by converting a markdown document and its contents to an HTML file and its syntax.
It could be useful as e.g a blog where you'd write a simple markdown file and the generator will take care of the rest.

This project will likely be discontinued, as it has just been a learning tool. But who knows? Maybe in the future.
